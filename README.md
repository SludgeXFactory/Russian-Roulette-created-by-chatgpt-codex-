# Russian Roulette Showdown

Russian Roulette Showdown is a browser-based Russian roulette game built with plain HTML, CSS, and JavaScript. It mixes high-risk turn strategy, hidden chamber information, stacked damage, power-up mind games, and a toxic dealer AI into a fast arcade-style duel experience.

The project includes both `Player vs Player` and `Player vs AI` modes, multiple dealer difficulties, a large power-up system, coin progression, a shop, VIP/dev systems, animated turn targeting, and heavy visual effects for shots, reloads, healing, and damage.

## Features

- 8-chamber Russian roulette gameplay
- `Player vs Player` and `Player vs AI`
- 5 dealer difficulty levels
- large power-up pool with offensive, defensive, control, and info items
- stacked `Shotgun Bullet` damage
- dealer AI that reacts to HP, bullet odds, known chamber info, and player inventory threats
- coin rewards, upgrades, and black market purchases
- mock premium shop and VIP systems
- registration and hidden dev mode for the `YeeDev` profile
- animated board effects, blood/heal screen tint, recoil, smoke, reload pulses, and turn indicators

## Tech

- HTML
- CSS
- JavaScript

## Run Locally

1. Open `index.html` in your browser.
2. Enter a name to create or load your local profile.
3. Start a new match and play.

## Project Structure

- `index.html`: site layout and game UI
- `style.css`: styling, animation, and visual effects
- `app.js`: game logic, AI, progression, and interactions

## Notes

- Profile data is stored in browser local storage.
- Premium checkout is a front-end mockup only and does not process real payments.
- The project is fully front-end and does not require a backend to run locally.
